このプレゼンテーションのデプロイ方法
GitHub Pagesでのデプロイ
  1. GitHubアカウントにログインし、新しいリポジトリを作成します
  2. このフォルダ内のすべてのファイルをリポジトリにアップロードします
  3. リポジトリの「Settings」→「Pages」を開きます
  4. Source を「main」ブランチに設定し、保存します
  5. 数分後、GitHubがウェブサイトをデプロイし、URLが表示されます
Netlifyでのデプロイ
  1. Netlifyアカウントを作成し、ログインします
  2. 「New site from Git」をクリックします
  3. GitHubを連携し、このファイルをアップロードしたリポジトリを選択します
  4. デプロイ設定を確認し、「Deploy site」をクリックします
  5. 数分後、サイトがデプロイされ、カスタムドメインの設定も可能になります
Vercelでのデプロイ
  1. Vercelアカウントを作成し、ログインします
  2. 「New Project」をクリックします
  3. GitHubを連携し、このファイルをアップロードしたリポジトリを選択します
  4. 「Import」をクリックし、デフォルト設定のまま「Deploy」をクリックします
  5. 数分後、サイトがデプロイされ、プロジェクトダッシュボードでURLが確認できます
